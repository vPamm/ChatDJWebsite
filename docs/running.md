# Running the program as a script

## Install go

https://go.dev/dl/

## Cloning and installing dependencies

Clone the project

```bash
  git clone https://github.com/vpamm/ChatDJ
```

Go to the project directory

```bash
  cd ChatDJ
```

Install dependencies
```go
  go get ./...
```

## Run the program!

```go
  go run .
```
